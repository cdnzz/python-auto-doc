Python Auto Doc
===============

Python Sphinx document auto generator

**Code Tree:**
<pre>
--autodoc.py
--doc
--src
  |
  --__init__.py
</pre>


**__init__.py at root of src must contains some property, For example:**
<code><pre>
__version__ = '0.1'
__author__ = ['Rebill']
__description__ = 'Style Guide for Python Code'
__dependent_3rd__ = {
    'gevent': 'http://www.gevent.org/'
}
</pre></code>
